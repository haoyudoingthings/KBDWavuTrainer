Tekken-style direction icons (optional).
Place PNG files named after directions to show icons in the input history instead of text:
  b.png, f.png, u.png, d.png, db.png, df.png, ub.png, uf.png
Recommended size: 24x24 or 32x32 pixels.
If any file is missing, the app falls back to text labels (e.g. "b", "db").
